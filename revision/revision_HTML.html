📄 Structure HTML5

Document HTML minimal

<!-- Déclaration HTML5 --> <!doctype html> <html lang="fr"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>Mon Site</title> <link rel="stylesheet" href="style.css"> </head> <body> <!-- Contenu ici --> </body> </html>
⚠️ Points importants :
  • <!doctype html> TOUJOURS en premier
  • lang="fr" pour indiquer la langue
  • charset="utf-8" pour les accents
  • viewport pour le responsive

Balises HTML essentielles

Catégorie Balises Usage
Titres <h1> à <h6> Hiérarchie des titres
Texte <p>, <em>, <strong> Paragraphes et emphase
Listes <ul>, <ol>, <li> Listes à puces / numérotées
Liens <a href="..."> Liens hypertextes
Images <img src="..." alt="..."> Images (alt obligatoire!)
Containers <div>, <section>, <article> Structuration du contenu

Éléments sémantiques HTML5

<header> - En-tête de page/section
<nav> - Navigation principale
<main> - Contenu principal (unique)
<article> - Contenu autonome
<section> - Section thématique
<aside> - Contenu annexe (sidebar)
<footer> - Pied de page

Tableaux HTML

<table> <thead> <tr> <th scope="col">Nom</th> <th scope="col">Prénom</th> </tr> </thead> <tbody> <tr> <td>Dupont</td> <td>Jean</td> </tr> </tbody> </table>
✅ Fusion de cellules :
  • colspan="2" : fusionne 2 colonnes horizontalement
  • rowspan="2" : fusionne 2 lignes verticalement

🎯 Sélecteurs CSS

Sélecteurs de base

Sélecteur Syntaxe Description
Élément p { } Tous les <p>
Classe .ma-classe { } Éléments avec class="ma-classe"
ID #mon-id { } Élément avec id="mon-id" (UNIQUE!)
Universel * { } Tous les éléments
❌ ERREUR FRÉQUENTE : Un ID doit être UNIQUE dans la page ! Pour plusieurs éléments, utilisez des classes.

Sélecteurs contextuels TRÈS IMPORTANT

Sélecteur Syntaxe Description
Descendant E F { } F inclus dans E (à n'importe quel niveau)
Enfant direct E > F { } F enfant direct de E
Adjacent E + F { } F juste après E (même parent)
Frère suivant E ~ F { } Tous les F après E (même parent)
Multiple E, F { } E ET F
/* Exemples pratiques */ article p { color: blue; } /* Tous les p dans article */ div > p { color: red; } /* p enfants directs de div */ h2 + p { font-weight: bold; } /* p juste après h2 */ h1 ~ p { color: gray; } /* Tous les p après h1 */

Sélecteurs d'attributs

Sélecteur Description
E[attr] E possédant l'attribut
E[attr="val"] Valeur exacte
E[attr*="val"] Contient "val"
E[attr^="val"] Commence par "val"
E[attr$="val"] Finit par "val"
/* Exemples pratiques */ a[href^="https"] { color: green; } /* Liens HTTPS */ img[src$=".png"] { border: 1px solid blue; } /* Images PNG */ a[href*="youtube"] { color: red; } /* Liens vers YouTube */

Pseudo-classes

États des liens :
a:link { } /* Non visité */ a:visited { } /* Visité */ a:hover { } /* Survolé */ a:active { } /* Cliqué */
Position :
li:first-child { } li:last-child { } li:nth-child(2n) { } /* pairs */ li:nth-child(odd) { } /* impairs */

Pseudo-éléments

p::first-letter { font-size: 2em; } p::first-line { font-weight: bold; } .important::before { content: "⚠️ "; } a[href^="http"]::after { content: " 🔗"; }

📐 Flexbox PRIORITÉ MAXIMALE

💡 Concept : Flexbox permet d'organiser des éléments en ligne ou en colonne avec un contrôle précis de l'alignement et de la répartition de l'espace.

Activation de Flexbox

/* Sur le PARENT (container) */ .container { display: flex; /* Active flexbox */ }
⚠️ Important : display: flex se met sur le PARENT, pas sur les enfants !

Propriétés du Container (Parent)

Propriété Valeurs Description
flex-direction row | column | row-reverse | column-reverse Direction de l'axe principal
justify-content flex-start | flex-end | center | space-between | space-around Alignement sur l'axe principal
align-items stretch | flex-start | flex-end | center Alignement sur l'axe secondaire
flex-wrap nowrap | wrap | wrap-reverse Retour à la ligne

Propriétés des Items (Enfants)

Propriété Description
flex-grow: 1 Facteur d'expansion (prend l'espace disponible)
flex-shrink: 1 Facteur de rétrécissement
flex-basis: auto Taille de base
flex: 1 Raccourci pour grow shrink basis

Cas d'usage fréquents

Centrer un élément :
.container { display: flex; justify-content: center; align-items: center; height: 100vh; }
Navigation horizontale :
nav { display: flex; justify-content: space-between; align-items: center; }

📱 Responsive Design

Media Queries

/* Mobile First : styles de base pour mobile */ body { font-size: 14px; } /* Tablette et plus (≥ 768px) */ @media screen and (min-width: 768px) { body { font-size: 16px; } } /* Desktop (≥ 1024px) */ @media screen and (min-width: 1024px) { body { font-size: 18px; } }

Breakpoints courants

Device Breakpoint Media Query
Mobile < 576px Styles par défaut
Tablette portrait ≥ 576px @media (min-width: 576px)
Tablette paysage ≥ 768px @media (min-width: 768px)
Desktop ≥ 992px @media (min-width: 992px)
Large desktop ≥ 1200px @media (min-width: 1200px)

Types de média

/* Pour l'écran */ @media screen { } /* Pour l'impression */ @media print { nav, aside { display: none; } body { color: black; background: white; } } /* Orientation */ @media (orientation: portrait) { } @media (orientation: landscape) { }
💡 Astuce : Utilisez l'approche "Mobile First" : commencez par les styles mobiles, puis ajoutez des media queries pour les écrans plus grands.
